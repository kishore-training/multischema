Welcome to Hospital Appointment multischema Application!



This is a demo of multischema hospital management application on cloud. A single instance of this application can handle multiple customers/tenants/hospitals using seperate database for each tenant. Customers will have to complete registration and provide details of their tenant admin user


This application enables entering and viewing the appointments of doctors working in a hospital.
